Markdown

Markdown is a lightweight markup language for creating formatted text using a plain-text editor. John Gruber created Markdown in 2004 as a markup language that is easy to read in its source code form. It is commonly used to format README files, for writing messages in online discussion forums, and to create rich text using a plain text editor.

Markdown is widely used in blogging, instant messaging, online forums, collaborative software, documentation pages, and readme files. GitHub, Stack Overflow, and other platforms use Markdown to format comments and content.

Markdown is a markup language that, in plain text, uses symbols like # or * to structure documents. Markdown's key design goal is readability — that the language be readable without looking like it has been marked up.
